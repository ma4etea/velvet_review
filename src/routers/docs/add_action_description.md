# Совершает действие над товаром
- Проверяет, является ли пользователь администратором компании (***доступны любые действия***) `company_role` -> `{{ admin_roles }}`.
  - Если не администратор тогда ***для каждого вида*** действия проверяется `stores_roles`:
    1. Действие `{{ action_1 }}`, пополнить запас товара, доступно для: `{{ roles_can_1 }}`.
    2. Действие `{{ action_2 }}`, продажа товара, доступно для: `{{ roles_can_2 }}`.
    3. Действие `{{ action_3 }}`, возврат проданного товара, доступно для: `{{ roles_can_3 }}`.
    4. Действие `{{ action_4 }}`, списание товара из запаса, доступно для: `{{ roles_can_4 }}`.
    5. Действие `{{ action_5 }}`, новая цена товара, доступно для: `{{ roles_can_5 }}`.
    6. Действие `{{ action_6 }}`, возврат товара поставщику, доступно для: `{{ roles_can_6 }}`.

- Возвращает идентификатор действия `id`
