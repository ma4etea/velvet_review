# Подтверждает регистрацию пользователя.

- Метод API разрешен для:
    - Администраторов, **разрешенные роли в компании**: `{{ admin_roles }}`.

- Если email пользователя на этот момент подтвержден, тогда **регистрация выполняется**.
- Если email еще не подтвержден, сообщение об ожидании подтверждение email.
