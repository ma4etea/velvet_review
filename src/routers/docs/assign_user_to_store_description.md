# Добавляет роль пользователя для определенного магазина

- Метод API разрешен для:
    - Администраторов, **разрешенные роли в компании**: `{{ admin_roles }}`.

- Доступные роли пользователей для магазина: `{{all_store_roles}}`.

- Запрещено назначать роль **администраторам компании**.
- Возвращает объект магазина с назначенными пользователями.

### ⚠️ После изменения роли необходимо обновить токен доступа: `accessToken`
