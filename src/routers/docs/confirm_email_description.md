# Подтверждает email нового пользователя

- Проверяет код подтверждения для email

- Подтверждение email:
    - Если регистрируется первый пользователь:
        - После подтверждения кода пользователь становится `owner`, **регистрация выполняется**.

    - Иначе регистрируется `member` второй и более пользователь:
        - Требуется подтверждение регистрации от администратора
            - если подтверждение уже есть, тогда **регистрация выполняется**.
            - если нет, возвращает сообщение об ***ожидании подтверждения регистрации администратором***
