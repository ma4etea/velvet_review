# Удаляет изображения для товара

- Метод API разрешен для:
    - Администраторов, **разрешенные роли в компании**: `{{ admin_roles }}`.
    - Пользователей, **разрешенные роли пользователя в конкретном магазине**: `{{ can_delete_image }}`.

- Нельзя удалить изображение, которое является главным `main_image` для товара.
- Возвращает **сообщение об успешном удалении**
