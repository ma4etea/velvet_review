# Получает действие с транзакциями и товарами, которые участвовали в этом действии.

- Проверяет, является ли пользователь администратором компании, **(разрешено все) роли в компании**: `{{ admin_roles }}`.

- Если не администратор тогда **проверяется доступ по роли пользователя в конкретном магазине**, разрешено для: `{{ can_get_actions }}`.

- Возвращает объект `action` со списком транзакций с товарами
