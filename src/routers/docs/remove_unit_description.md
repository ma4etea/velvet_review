# Удаляет товар с картинками, только если у товара нет транзакций
- Метод API разрешен для:
    - Администраторов, **разрешенные роли в компании**: `{{ admin_roles }}`.
    - Пользователей, **разрешенные роли пользователя в конкретном магазине**: `{{ can_delete_unit }}`.

- Возвращает **сообщение об успешном удалении**.
