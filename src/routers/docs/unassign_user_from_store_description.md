# Удаляет роль пользователя для определенного магазина

- Метод API разрешен для:
    - Администраторов, **разрешенные роли в компании**: `{{ admin_roles }}`.

- Запрещено удалять роль **администраторам компании**.
- Возвращает сообщение об успешном выполнении.

### ⚠️ После изменения роли необходимо обновить токен доступа: `accessToken`
