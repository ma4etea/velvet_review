# Изменяет роль пользователя в компании.

- Метод API разрешен для:
    - Ролей компании: `{{ can_update_company_role }}`.

- Можно изменить **роль пользователя в компании** на: `{{ company_roles }}`
- Запрещено изменять роль:
    - Самому себе.
    - Владельцу компании.

- Возвращает обновленный объект пользователя.

### ⚠️ После изменения роли необходимо обновить токен доступа: `accessToken`
