# Загружает изображения для товара

- Метод API разрешен для:
    - Администраторов, **разрешенные роли в компании**: `{{ admin_roles }}`.
    - Пользователей, **разрешенные роли пользователя в конкретном магазине**: `{{ can_upload_images }}`.

- Первое добавленное изображение автоматически становится `main_image`.
- Поддерживаемые форматы изображения: `{{ supported_extensions }}`
- Максимальное количество изображений для одного товара: `{{ max_images }}`

- Возвращает **сообщение: изображения для товара получены и находятся в обработке**.
